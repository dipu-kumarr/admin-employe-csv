CSV Format Guidelines

Your CSV file should contain the following columns in this order:

  • Name - The name associated with the record
  • Title - The title of the record
  • Description - A detailed description of the record

First row should contain headers. Example:

Name,Title,Description
John Doe,Project Report,This is a report about the project status
Jane Smith,Meeting Notes,Notes from the client meeting on 2022-01-15
